# ISTQB CTFL Study Notes

Study notes for the ISTQB Certified Tester Foundation Level exam. I am currently preparing for this certification.

## About the exam

ISTQB CTFL is the most widely recognised entry level certification in software testing. It covers testing fundamentals, testing throughout the software lifecycle, static testing, test analysis and design, test management, and tool support for testing.

I am working through the official ISTQB CTFL syllabus and will document key concepts, definitions, and tips here as I study.
